# HR Analytics Dashboard using Power BI

## Project Overview
An interactive HR Analytics Dashboard developed using Microsoft Power BI to analyze employee attrition, demographics, department-wise distribution, and job satisfaction.

## Features
- Total Employees
- Average Age
- Average Monthly Income
- Attrition Count
- Employees by Gender
- Employees by Department
- Employees by Age Group
- Attrition by Job Role
- Attrition by Marital Status
- Attrition by Education Field
- Attrition by Business Travel
- Attrition by Years at Company
- Job Satisfaction by Job Role

## Tools Used
- Microsoft Power BI
- Excel
