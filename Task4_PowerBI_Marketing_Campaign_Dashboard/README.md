# Marketing Campaign Dashboard - Power BI

## Project Overview
This project analyzes a marketing campaign dataset using Microsoft Power BI. The dashboard provides insights into customer demographics, purchasing behavior, campaign performance, and web activity.

## Dashboard Features
- Total Customers
- Average Income
- Average Recency
- Average Web Purchases
- Customers by Education
- Customers by Marital Status
- Campaign Acceptance
- Purchase by Product
- Purchases by Purchase Channel
- Education Slicer

## Tools Used
- Microsoft Power BI
- CSV Dataset

## Dataset
marketing_campaign.csv
